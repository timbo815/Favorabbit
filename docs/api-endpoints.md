# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`



## JSON API

### Favors

- `GET /api/favors`

- `POST /api/favors`
- `GET /api/favors/:id`
- `PATCH /api/favors/:id`
- `DELETE /api/favors/:id`

### Categories

- `GET /api/categories`
